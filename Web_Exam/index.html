Go To Sleep

Description

Go To Sleep is an app written in Java using Android Studio. The main purpose of this app is to help people fix their sleeping schedule, the user chooses a time range and the settings that will work during the set time thus preventing the user from using their phone. This app can also be used to limit somebodies phone usage at certain times of the day.

What I learned

  • The Basics of Java
  • Android Studio Development Environment
  • Improved my Understanding of UI/UX Design

Technologies used

  • Java
  • Android Studio

Car Tuner

Description

This project is unfinished. The point of this app was to create software that would communicate with a car through a COM port, the app would gather data from the car and output that in the software, then the data can be manipulated and imported back in to the car. What I learned

Technologies used

  • Unity
